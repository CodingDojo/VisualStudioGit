The Git Macro allows to commit from within Visual Studio. It opens an inputbox to enter the commit message and commits to git.

1) Open Visual Studio Macro IDE:
    Tools -> Macros -> Macro IDE
2) Add Macro to your "MyMacros":
    In "Microsoft Visual Studio Macros": Project -> Add existing item -> Select "Git.vb"
3) Bind to key:
    In Visual Studio -> Tools -> Options -> Environment -> Keyboard -> Select Macros.MyMacros.Git.Commit
    Press shortcut keys: <<Select your shortcut>> -> Assign -> OK
